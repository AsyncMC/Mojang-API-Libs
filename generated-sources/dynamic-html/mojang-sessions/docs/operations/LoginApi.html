This is the API

getPlayerProfile

This will return the player's username plus any additional information about them (e.g. skins). This has a much stricter rate limit: You can request the same profile once per minute, however you can send as many unique requests as you like.

URL

https://sessionserver.mojang.com/session/minecraft/profile/{stripped_uuid}

HTTP Method

GET

Response Type

PlayerProfile

Parameters

  • Path: stripped_uuid String

    The player UUID without hyphens

  • Query: unsigned Boolean

    When set to false, the signature data are sent
