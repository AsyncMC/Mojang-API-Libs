This is the API

changePlayerSkin

This will set the skin for the selected profile, but Mojang's servers will fetch the skin from a URL. This will also work for legacy accounts.

URL

https://api.mojang.com/user/profile/{stripped_uuid}/skin

HTTP Method

POST

Response Type

Parameters

  • Path: stripped_uuid String

    The player UUID without hyphens

  • Form: url String(string)

    The URL which Mojang servers will download and apply the skin

  • Form: model SkinModel(SkinModel)

resetPlayerSkin

URL

https://api.mojang.com/user/profile/{stripped_uuid}/skin

HTTP Method

DELETE

Response Type

Parameters

  • Path: stripped_uuid String

    The player UUID without hyphens

uploadPlayerSkin

This uploads a skin to Mojang's servers. It also sets the users skin. This works on legacy counts as well.

URL

https://api.mojang.com/user/profile/{stripped_uuid}/skin

HTTP Method

PUT

Response Type

Parameters

  • Path: stripped_uuid String

    The player UUID without hyphens

  • Form: file File(file)

    The skin image in PNG format

  • Form: model SkinModel(SkinModel)
